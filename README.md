# App de Recettes 🍽️

Bienvenue sur l'application de recettes inspirée par la cuisine haïtienne et internationale.

## 📂 Catégories Incluses
- Petit Déjeuner (Haïtien & International)
- Déjeuner (Haïtien & International)
- Dîner (Haïtien & International)
- Desserts
- Boissons

## 🚀 Démarrage

```bash
npm install
npm run dev
```

## 🛠️ Tech Utilisée
- Remix.run
- React
- TailwindCSS
